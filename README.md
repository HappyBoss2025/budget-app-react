# BudgetApp - Dashboard de Gestion Financière

Une application web moderne et responsive développée avec **React.js** permettant de suivre en temps réel l'état de ses finances personnelles (Recettes, Dépenses et Solde).

## 🚀 Fonctionnalités
* **Tableau de bord dynamique** : Visualisation instantanée du solde total, des recettes et des dépenses.
* **Gestion des transactions** : Ajout et suppression de flux financiers avec typage (Dépense/Recette).
* **Visualisation de données** : Intégration d'un graphique circulaire (Pie Chart) pour l'analyse de la répartition financière.
* **Interface Responsive** : Design épuré utilisant Bootstrap et Lucide Icons pour une expérience utilisateur fluide.

## 🛠️ Technologies utilisées
* **Frontend** : React.js (Hooks: useState, useEffect)
* **Styling** : Bootstrap 5, CSS3
* **Graphiques** : Chart.js & React-chartjs-2
* **Icons** : Lucide-React

## 📌 Architecture Technique
L'application repose sur une gestion d'état centralisée via les **React Hooks**, assurant une réactivité optimale de l'interface sans rechargement de page. 

> **Note :** Cette version actuelle est une démonstration "Frontend-Only" utilisant un état local pour la persistance temporaire des données. L'architecture est conçue pour être facilement connectée à une API REST (via Axios ou Fetch) pour une persistence de données sur base de données.
